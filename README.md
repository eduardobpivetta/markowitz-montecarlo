# 📈 Markowitz + Monte Carlo (Portfolio Simulator)

Aplicação interativa em Streamlit para otimização de carteiras com base na Teoria de Markowitz e simulação de cenários futuros usando Monte Carlo.

## 🚀 Funcionalidades

### 🏆 Otimização de Carteira (Markowitz)
- Simulação de milhares de carteiras aleatórias
- Cálculo da fronteira eficiente
- Identificação da carteira com maior índice de Sharpe
- Visualização risco vs retorno

### 📊 Comparação com Benchmarks
- CDI e IPCA (via Banco Central)
- IBOV, S&P 500, IFIX, entre outros (via Yahoo Finance)

### 🎲 Simulação de Monte Carlo
- Projeção do crescimento do capital ao longo do tempo
- Simulações com correlação entre ativos (Cholesky)
- Intervalos de confiança (P50, P5, P1)
- Probabilidade de lucro
